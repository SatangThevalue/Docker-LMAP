## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/SatangThevalue/docker-lmap

# Access
$ cd docker-lmap

# Run the project
$ docker-compose up -d

# The server will initialize in
PHP 
        HTTP port 80
        HTTPS port 443
mariadb 
        port 3306
        password docker
phpmyadmin
        port 8081
        PMA_USER root
        PMA_PASSWORD docker
```
